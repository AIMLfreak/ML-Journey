## 🌐 GitHub Pages Deployment

### Enable GitHub Pages

1. Push your repository to GitHub
2. Open:

   * Settings
   * Pages
3. Under **Build and deployment**

   * Source → Deploy from branch
   * Branch → `main`
   * Folder → `/root`
4. Save

Your site will be available at:

```bash
https://aimlfreak.github.io/<repository-name>/
```
